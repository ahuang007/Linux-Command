# Linux mcd 命令

返回 [Linux 命令大全](https://ahuang007.github.com/Linux-Command)

Linux mcd为mtools工具指令，可在MS-DOS文件系统中切换工作目录。若不加任何参数，则显示目前所在的磁盘与工作目录。

### 语法

```
mcd [msdosdirectory]
```

### 实例

变更目前工作目录到 a: emp 中。

```
mcd a: emp
```

传回目前工作目录。

```
mcd
```
