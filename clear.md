# Linux clear 命令

返回 [Linux 命令大全](https://ahuang007.github.com/Linux-Command)

Linux clear 命令用于清除屏幕。

### 语法

```
clear
```

清屏

```
# 上一屏

drwxrwxr-x  2 huanglijun huanglijun   91 8月  19 10:10 test
[huanglijun@localhost ~]$ clear

# 下一屏
```
