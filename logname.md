# Linux logname 命令

返回 [Linux 命令大全](https://ahuang007.github.com/Linux-Command)

Linux logname 命令用于显示用户名称。

执行 logname 指令，它会显示目前用户的名称。

### 语法

```
logname [--help][--version]
```

**参数**：

- --help 　在线帮助。
- --vesion 　显示版本信息。

### 实例

显示登录账号的信息：

```
[huanglijun@localhost ~]$ logname
huanglijun
```
