# Linux pwd 命令

返回 [Linux 命令大全](https://ahuang007.github.com/Linux-Command)

Linux pwd 命令用于显示工作目录。

执行 pwd 指令可立刻得知您目前所在的工作目录的绝对路径名称。

### 语法

```
pwd [--help][--version]
```

**参数说明:**

- --help 在线帮助。
- --version 显示版本信息。

### 实例

查看当前所在目录：

```
[huanglijun@localhost ~]$ pwd
/home/huanglijun
```
