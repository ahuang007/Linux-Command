# Linux mrd 命令

返回 [Linux 命令大全](https://ahuang007.github.com/Linux-Command)

Linux mrd 命令用于删除 MS-DOS 文件系统中的目录。

mrd为 [mtools](https://github.com/ahuang007/Linux-Command/blob/master/mtools.md)工具指令，模拟 MS-DOS 的 rd 指令，可删除 MS-DOS 的目录。

### 语法

```
mrd [目录...]
```
